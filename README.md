# GitHub Jobs API

API utilizada pelo projeto GitHub Jobs.  
Fornece dados fictícios de vagas de emprego para fins de estudo e desenvolvimento.

## Descrição

A GitHub Jobs API é uma API simples que disponibiliza endpoints para listar vagas,
visualizar detalhes e aplicar filtros básicos.  
Ela foi criada apenas para demonstrar o consumo de API no projeto principal.
